uint32_t lzma_options_delta::dist

Delta distance.

With the only currently supported type, LZMA_DELTA_TYPE_BYTE, the distance is as bytes.

Examples:

  • 16-bit stereo audio: distance = 4 bytes
  • 24-bit RGB image data: distance = 3 bytes
