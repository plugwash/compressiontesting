Set a compression preset to \doxyref{lzma\-\_\-options\-\_\-lzma}{p.}{structlzma__options__lzma} structure. 

0 is the fastest and 9 is the slowest. These match the switches -\/0 .. -\/9 of the xz command line tool. In addition, it is possible to bitwise-\/or flags to the preset. Currently only L\-Z\-M\-A\-\_\-\-P\-R\-E\-S\-E\-T\-\_\-\-E\-X\-T\-R\-E\-M\-E is supported. The flags are defined in \doxyref{container.\-h}{p.}{container_8h}, because the flags are used also with \doxyref{lzma\-\_\-easy\-\_\-encoder()}{p.}{container_8h_acbdad999c544872f0f5d242f0d1a4ed4}.

The preset values are subject to changes between liblzma versions.

This function is available only if L\-Z\-M\-A1 or L\-Z\-M\-A2 encoder has been enabled when building liblzma.

\begin{DoxyReturn}{Returns}
On success, false is returned. If the preset is not supported, true is returned. 
\end{DoxyReturn}
